What I do

Design/development

Since I have always been very interested in awesome looking web designs and animations, and envied the people who could make these cool things work through programming, I decided to do something about it in early 2021.

I did some research on the internet and found out that there are a lot of very comprehensive tutorials out there for anyone who wants to teach themselves how to program. So I started following and coding HTML and CSS tutorials and trying out front-end coding challenges to learn the basics of web development.

However, the learning does not stop when the tutorial is over. That's why I keep checking back every day, discovering and learning new designs and developments to build a set of skills and feel more comfortable in web development every time. Currently, I am still building my HTML and CSS skills, and I recently started JavaScript as well. I have never been so eager to dive deep into the challenging and interesting world of functions, arrays, variables, loops, ...

Freestyle illustrator

As a fan of manga, anime, and comics, drawing superheroes and villains was one of my favorite things to do growing up. I think that was one of the main triggers for my interest in design in general.

So learning to work with different (digital) tools to design and draw was an indispensable step for me, as the digital world offers so many opportunities to get creative and turn imagination into illustrations. Getting creative with these digital tools is a never-ending adventure as I discover and learn new features and skills each time as the tools offer more and more opportunities to reach the next level of creativity.
